A school iPad removal request system.  
Developed as a **Fullstack Web Application** with role-based access control.

## Project Overview

This system is designed for  
**Parents / Guardians and IT staff**  
to manage iPad removal requests in a clear, structured, and traceable process.

The system supports:
- Creating requests
- Step-by-step approval workflow
- Restricted editing based on request status
- Printing PDF documents for real-world use

## User Roles

This system is **used by IT staff only**.
Review requests, approve or reject them, add IT notes, and print PDF documents |
> The system uses role-based access control.
